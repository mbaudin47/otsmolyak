# otsmolyak module

Examples
  Template examples can be found here: C:\Python27\Lib\site-packages\otsmolyak\examples\otsmolyak

Installation
  Silent install can be done with the switch /S (e.g. openturns-otsmolyak-0.0-py27.exe /S).
  To test otsmolyak module installation, in an OpenTURNS python console type:
    import otsmolyak
    print otsmolyak.__version__

Uninstallation
  OtSmolyak module will be automatically uninstalled at OpenTURNS removal.
  OtSmolyak module can be removed from OpenTURNS with uninst-template.exe uninstaller.
  Silent uninstall can be done with the switch /S.
